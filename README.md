sOS
===

Versión 1

Descripción
-----------
sOS es una GUI que busca ser user-friendly para que los usuarios puedan buscar y ejecutar aplicaciones/programas instalados en su PC con CraftOS.

La idea surge de la necesidad de distribuir aplicaciones, ejecutables en CraftOS del mod de Minecraft CC:Tweaked, en los PCs de mis amigos en Minecraft (que nunca han usado una terminal de Linux).
Puesto que la interfaz de CraftOS es una CLI, mis amigos no sabrían ejecutar las apps que desarrolle para nuestro servidor de Minecraft.

En principio la interfaz debería proporcionar funciones básicas, como un dock de aplicaciones, un menú de notificaciones, ajustes de colores y tamaño de letra, soporte para instalar/desinstalar/ejecutar aplicaciones y marcarlas/desmarcarlas como favoritas.
Desde la v1 de sOS, se proporciona la opción de controlar multiples monitores y altavoces conectados directamente con el PC.

sOS incluye su propia libreria de funciones para trabajar con textos en multiples pantallas.

Objetivos
---------
La v1 de sOS estará completa cuando:
 - Se termine la interfaz principal para PCs.
 - Se optimice la interfaz para PCs de bolsillo.
 - Se proporcione control total sobre todo tipo de periféricos que pueden estar, o no, directamente conectados al PC.
 - Se den opciones de apagado y encendido seguros para el propio sOS.
 - Se proporcione un método para instalar/desinstalar aplicaciones de forma segura.
 - Se garantice el paralelismo para la impresión de texto en multiples pantallas desde la propia librería de gráficos.
 - Se dejen de incluir archivos de configuración de sOS del usuario en los commits.
 - Se mejoren los mecanismos de guardado de configuraciones del usuario y la serialización de las tablas que los contienen.
 - Se disponga de documentación para cada función y parte del código.

Disclaimer
----------
Este es un proyecto que desarrollo en mi tiempo libre y como hobby, lo hago también para mejorar mi lógica de programación y de resolución de problemas.

#
##### English
Description
-----------
sOS is a GUI designed to be user-friendly so that users can search for and execute applications/programs installed on their PC with CraftOS.

The idea arises from the need to distribute applications, executables in CraftOS from the Minecraft mod CC:Tweaked, to my friends' PCs in Minecraft (who have never used a Linux terminal). Since the CraftOS interface is a CLI, my friends wouldn't know how to run the apps I develop for our Minecraft server.

Initially, the interface should provide basic functions such as an application dock, a notification menu, color and font size settings, support for installing/uninstalling/running applications, and marking/unmarking them as favorites. Since v1 of sOS, the option to control multiple monitors and speakers connected directly to the PC is provided.

sOS includes its own library of functions to work with texts on multiple screens.

Objectives
----------
sOS v1 will be complete when:
 - The main interface for PCs is finished.
 - The interface is optimized for pocket PCs.
 - Full control is provided over all types of peripherals that may or may not be directly connected to the PC.
 - Safe shutdown and startup options are provided for sOS itself.
 - A method for installing/uninstalling applications securely is provided.
 - Parallelism is ensured for text printing on multiple screens from the graphics library itself.
 - User sOS configuration files are no longer included in commits.
 - Mechanisms for saving user configuration and serializing the tables containing them are improved.
 - Documentation is available for each function and part of the code.
